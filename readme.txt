To improve accessibility of the website, the following items were modified accordingly to achieve the best result.

1. contrast
  Verified contrast using https://webaim.org/resources/contrastchecker/. Changed "blue" and "grey" background to for white and black text.
2. ease of clickability
  Added padding for <nav> and <a>.
3. zoom, responsive design principles
  Implemented css grid box. Verified the zoom in the range from 75% to 200%.
4. skip link
  Added <a href="#main-content" class="hidden">Skip to main content</a>
5. screenreader only content
  Added <span class="sr-only">these links opens in a new tab</span> to footer nav
6. tab index
  Modified <a href="index.html" tabindex="1">About</a>
7. image alt tags
  Verified all images contain alt attributes.
